# Lucee Rest API

## Untuk Menjalankan

```bash
box server start
```

## Untuk Menghentikan

```bash
box server stop
```

## Untuk melakukan restart

```bash
box server restart
```

## Menhapus Lucee Server

```bash
box server forget
```

# Contoh acess

```
http://localhost:8888/rest/em/user/say
```
